# Week-10-Data-Preprocessing-Feature-Engineering
#  Customer Churn Prediction - Week 10

##  Project Overview
This project focuses on preprocessing customer data, performing feature engineering, and building a machine learning model to predict customer churn.

---

##  Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

---

##  Dataset
Customer churn dataset containing:
- Tenure
- Monthly Charges
- Total Charges
- Contract Type
- Payment Method
- Churn Status

---

##  Steps Performed

### 1. Data Preprocessing
- Cleaned column names
- Handled missing values
- Converted data types

### 2. Encoding Techniques
- Label Encoding (binary variables)
- One-Hot Encoding (categorical variables)

### 3. Feature Scaling
- StandardScaler applied to numerical features

### 4. Feature Engineering
Created new features:
- Avg Monthly Value
- Tenure Group
- Charges per Tenure
- High Value Customer Flag
- Long-Term Customer Flag

### 5. Outlier Handling
- Used IQR method to remove outliers

### 6. Model Building
- Logistic Regression model

---

##  Model Evaluation

- Accuracy: ~0.75 - 0.80
- Precision: Good
- Recall: Moderate
- F1 Score: Balanced
